# 함수형 업데이트 방식
- 함수형 업데이트 방식을 활용하여 의존성 배열에 state가 담기지 않도록 한것이 핵심!

```
  const onChange = useCallback(e => {
    const { name, value } = e.target;
    setInputs(inputs => ({
      ...inputs,
      [name]: value
    }));
  }, []);
```
