
1. 상수화 , 이름으로 의미부여

```
{index === 4}
```

가 아니라 
```
const ADExoposureIndex = 4
{index === ADExoposureIndex}

광고가 특정한 인덱스에 노출되어야하는 것을 상수명으로 줘야한다. 
```

-> 필요없는 변수는 사용하면 안된다.
