## 🏁 프로젝트 실행 방법

```
1. 의존성 패키지를 설치합니다.
```zsh
$ npm install
```
2. 리눅스와 윈도우 husky 충돌 방지 후 사용을 위해 설치합니다. 
```zsh
$ chmod ug+x .husky/* 
```
3. 프로젝트를 실행합니다.
```zsh
$ npm start
```
